# 動的スピーカー管理機能実装

## 解決したいこと

現在のスピーカーセレクターは固定で A-E の 5 人のスピーカーに限定しようと思っていたが、実際に使ってみるといろんな speaker を登場させるニーズがあった

## 方針

- 最初は A のみ表示し、必要に応じて動的にスピーカーを追加する方式にすれば、より直感的で柔軟な UI になる
- "New Speaker"オプションを通じてスピーカーを追加することで、必要な分だけスピーカーを管理できる
